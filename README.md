# Kindergarten
Автоматизированная информационная система непосредственной образовательной деятельности дошкольного образовательного учреждения

3. Техническое задание и исходные данные к работе	

Объект автоматизации: организация образовательной деятельности (НОД) государственного казённого дошкольного образовательного учреждения (ДОУ) детского сада № 63 «Золотая рыбка».
Пользователи: заместитель заведующего по воспитательно-методической работе и педагоги ДОУ. 
Входные данные: персональные данные педагогов и воспитанников, результаты проведения занятий в группе, в виде уровня развития воспитанников, результаты проведения мониторинга, нормы и требования СанПиН об организации образовательной деятельности в ДОУ, рекомендации учебной программы «От рождения до школы» под редакцией Н.Е.Вераксы, результаты участия педагогов и воспитанников в различных мероприятиях (дипломы, грамоты, сертификаты и т.д.), конспекты НОД.
Функции системы:
1)	ведение базы данных педагогов и воспитанников;
2)	учёт сведений о наградных документах;
3)	учёт сведений о результатах педагогического мониторинга;
4)	учёт требований и рекомендаций СанПиН и учебной программы;
5)	ведение картотеки конспектов НОД;
6)	формирование отчётности:
- по результатам методической активности педагогов за выбранный период;
- по результатам образовательной деятельности воспитанников;
- по результатам проведения педагогического мониторинга по группам и по ДОУ;
7)	выгрузка наградных документов в оценочный лист эффективности деятельности педагога;
8)	составление и ведение портфолио воспитанника;
9)	составление расписания НОД в группах на учебный год;
10)	составление перспективного планирования в рабочих программах педагогов;
11)	анализ результатов педагогического мониторинга для составления плана индивидуальных занятий.
Выходные данные: список педагогов и воспитанников, аналитические отчеты по педагогическому мониторингу по группам и по ДОУ, информация об уровне развития воспитанников в группах, расписание НОД по группам ДОУ, список наградных документов за отчетный период для педагогов и воспитанников, портфолио воспитанника, отчёт по методической активности педагогов ДОУ, картотека конспектов НОД.
Средства разработки: система управления базами данных MySQL Server, язык программирования Java.
